# Security policy

## Reporting a vulnerability

Please report security vulnerabilities privately through
[GitHub Security Advisories](https://github.com/stevencarpenter/token-auditor/security/advisories/new).
Do not open a public issue for a vulnerability.

Include the affected version or commit, reproduction steps, and the impact if
known. Please redact prompts, session logs, credentials, and local filesystem
paths from reports.

## Supported versions

Security fixes are applied to the latest release on the default branch. Older
versions may not receive backports.
